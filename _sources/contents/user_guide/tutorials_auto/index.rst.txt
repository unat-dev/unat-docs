*****************************
Creating Uncertain Quantities
*****************************

These tutorials present the ``UncertainQuantity`` type, which can be certainly
considered the entry point of the UnAT. It is mostly intended to the user that
have measurements collected from an experiment and now needs to handle it in a
simple fashion. Here, you will learn UnAT most basic tools to visualise your
samples and bla bla bla.
